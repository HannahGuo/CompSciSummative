Programming Language - Python 3.6.5
Problem Description - Roboto is a game where the player plays as a light-coloured robot (called Roboto) against a dark-
coloured robot (called Dark Roboto). The goal is for the player to hit Dark Roboto as many times as possible
before they get hit by one of Dark Roboto's shots.
Program Assumptions - This program requires a computer that has Python 3.6.5 installed. It has been tested using the
Windows 10 OS, but should also be compatible on Mac OS. It also requires the pygame module to be installed, and all
of the files that are in the Github Repository to be present.
